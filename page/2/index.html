#SQL基础

这篇文章是为了总结巩固SQL语句的基本用法

##数据库基础
SQL是用于访问和处理数据库的标准的结构化查询语言。

数据库（database）是保存有组织的数据的容器。

DBMS - 数据库管理系统（Database Management System。

表（table）是某种特定类型数据的结构化清单。

列（colomn)是表中的一个字段，所有表是由一个或多个列组成的。

数据类型（datatype）是所允许的数据的类型。每个列有其对应的数据类型。

行（row）是表中的一个记录。

主键（primary key）唯一标识表中每个记录的列。（总是需要定义）

##SQL基础语句
SELECT语句用于从表中选取数据

SELECT 列 FROM 表

DISTINCT用于返回唯一不同的值

SELECT DISTINCT 列 FROM 表

WHERE用于有条件的从表中选取数据

SELECT 列 FROM 表 WHERE 列 运算符 值

AND 和 OR 可在 WHERE 子语句中把两个或多个条件结合起来

ORDER BY语句用于根据指定的列对结果集进行排序，默认升序，加DESC降序

INSERT INTO语句用于向表格中插入新的行

INSERT INTO 表 VALUES（值, ...)

Update语句用于修改表中的数据

UPDATE 表 SET 列 = 新值 WHERE 列 = 某值

DELETE 语句用于删除表中的行

DELETE FROM 表 WHERE 列 = 值

##SQL高级语句
TOP 子句用于规定要返回的记录的数目（大型表）

SELECT TOP number|percent 列 FROM 表

LIKE 操作符用于在 WHERE 子句中搜索列中的指定模式（通配符）

%：任意字符出现任意多次
_:单个字符出现任意多次
[]:字符集，匹配括号中的一个

In 用在WHERE语句中规定多个值，BETWEEN用在WHERE语句中规定范围

AS可以用来为列名称和表名称指定别名（代替过长的名称）

join 用于根据两个或多个表中的列之间的关系，从这些表中查询数据

SELECT ... FROM 表1 （INNER）JOIN 表2 ON 表1.列=表2.列
SELECT ... FROM 表1，表2 WHERE 表1.列=表2.列（等价）

LEFT JOIN: 即使右表中没有匹配，也从左表返回所有的行

RIGHT JOIN: 即使左表中没有匹配，也从右表返回所有的行

FULL JOIN: 只要其中一个表中存在匹配，就返回行

UNION 操作符用于合并两个或多个 SELECT 语句的结果集

SELECT ... UNION SELECT ...

SELECT INTO从一个表中选取数据插入另一个表中（备份或存档）

SELECT ... INTO 新表 IN 新数据库 FROM...

CREATE DATABASE/TABLE 用于创建数据库/表

约束（CONSTRAINTS）

NOT NULL: 不能为NULL
UNIQUE: 不能重复
PRIMARY KEY： 主键，有且只有一个，NOT NUll and UNIQUE
FOREIGN KEY： 外键，链接两个表
              FOREIGN KEY（列）REFERENCES 外表（列）
CHECK： 约束列中值的范围

DEFAULT向列中插入默认值

CREATE INDEX用于在表中创建索引。可以更快地查找数据

CREATE INDEX 索引名 ON 表 (列)

DROP 可以删除索引、表和数据库

ALTER TABLE用于在已有的表中添加、修改或删除列

ALTER TABLE 表 ADD/DROP 列
    注意这里是对列操作，INSERT是对行（记录）进行添加

Auto-increment （自增）会在新记录插入时生成一个唯一的数字

VIEW（视图）是基于 SQL 语句的结果集的可视化的表

CREATE VIEW 视图名 AS （SQL语句）
    相当于把一大串SQL语句封装成一个视图，简洁高效

##SQL数据类型

INT存放整数值

FLOAT 单精度浮点数

CHAR(size) 保存固定长度的字符串

VARCHAR(size) 保存可变长度的字符串

TEXT 存放最大长度为 65,535 个字符的字符串。

ENUM(x,y,z,etc.) 允许你输入可能值的列表。

DATE存放日期和时间类型

…

##SQL函数
SQL 拥有很多可用于计数和计算的内建函数，分为Aggregate函数（面向一系列的值，并返回一个单一的值）和Scalar 函数（面向单一的值，返回单一的值）

AVG（）：求平均值
COUNT() 函数返回匹配指定条件的行数
FIRST() 函数返回指定的字段中第一个记录的值，（可先使用 ORDER BY 排序）
LAST() 函数返回指定的字段中最后一个记录的值
MAX（） 函数返回一列中的最大值
MIN（） 函数返回一列中的最小值
SUM（） 函数返回数值列的总数（总额）
GROUP BY 列 用于结合合计函数，根据一个或多个列对结果集进行分组
HAVING 是与合计函数一起使用的WHERE语句
UCASE（） 函数把字段的值转换为大写
LCASE（） 函数把字段的值转换为小写
MID（） 函数用于从文本字段中提取字符
LEN（） 函数返回文本字段中值的长度
ROUND（） 函数用于把数值字段舍入为指定的小数位数
NOW（） 函数返回当前的日期和时间
FORMAT（） 函数用于对字段的显示进行格式化
